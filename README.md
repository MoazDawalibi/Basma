# Basma Website

React, TypeScript, and Vite implementation of the Basma marketing website.

## Commands

```bash
npm run dev        # local development server
npm run lint       # static lint checks
npm run typecheck  # TypeScript checks
npm run build      # production build
npm run preview    # preview the production build
```

## Source structure

- `src/app` — application entry composition
- `src/components/ui` — reusable presentation primitives
- `src/features` — feature-owned pages, components, and content
- `src/styles` — global reset and design tokens
- `src/assets` — bundled image assets
